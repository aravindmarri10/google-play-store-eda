# Google Play Store App EDA

## Objective
Analyze Google Play Store app data and user reviews to understand factors influencing app success, installs, ratings, and user engagement.

## Dataset
- Google Play Store App Data
- User Review Data (Sentiment)

## Key Insights
-Optimize App Size
-Free apps scale significantly better than paid apps.
-Apps rated “Everyone” or “Everyone 10+” reach larger user bases.
-High-install apps still receive negative reviews—continuous improvement is critical.


## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
