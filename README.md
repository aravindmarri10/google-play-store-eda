# Google Play Store App EDA

## Objective
Analyze Google Play Store app data and user reviews to understand factors influencing app success, installs, ratings, and user engagement.

## Dataset
- Google Play Store App Data
- User Review Data (Sentiment)

## Key Insights
- User engagement strongly drives installs
- Free/freemium apps scale better
- App size and content rating impact adoption

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
